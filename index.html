Hiya, my name is Kelli; I've worked at my share of software companies over the last six years doing a little bit of this and that (tech support, seo, data hosting ops)... I like to think, I'm good at "getting in, where i fit in".

Now, during my time in Tech, I've also taken several bootcamp courses and dabbled with web development. Front-end web development is the area I enjoy exploring the most. My intention with this blog is to explore front-end development more in-depthly, and more specifically to test out some of the frameworks that can assist with the development process.

In other words, this space is also for rambling about my front-end development journey.

Hopefully, after my aforementioned spiel, you will decide to stick around for a little while longer.

Let's talk features

Bulma provides awesome features like:


Flexbox based: Makes creating grid items and vertically aligned things really easy.

Responsive: Mobile first framework similar to Bootstrap

Solid looking foundation: There is typography, buttons, tables, forms, and some basic CSS

Many components: Provides layouts, a vertical alignment level, and media objects
